How To Install

  • Download Theme
  • Unzip ( 7zip recommended )
    using password sun
  • Copy files to
    [Omega Install Path]\YGO Omega_Data\Files\Backgrounds
    and paste on the folder you want.
    example in Lobby folder for Lobby Theme
  • Disable Static background for effects.
  • Some theme may not contain music
    as those are copyrighted
